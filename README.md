# Dracus Research

Portal público de André Maia Martins / Dracus Research.

## Estado

`DRS-WEB-0.1` — fundação curatorial em desenvolvimento.

Esta branch contém a primeira interface institucional e uma cabine local para edição de letreiros, maturidade editorial e instruções de mudança. Alterações na cabine não publicam automaticamente.

## Princípios

- hipóteses não são apresentadas como resultados validados;
- a maturidade científica deve permanecer visível;
- o conteúdo reservado não integra o portal público;
- Lola Psyca é o sistema, Professor Lola sua função pedagógica, Medusa seu avatar e Dragão sua função crítica;
- publicação e mudanças de domínio exigem autorização humana.
